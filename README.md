## DemoApp LandmarkBook
The LandmarkBook app lists important landmarks of cities and allows you to view their photos. 
### App Contains
1. Page transition
2. Listing
3. Image Viewing

### In this demo app I used and practiced
- TextView
- ImageView
- ViewBingding
- ListView
- Mapping
- Adapter
